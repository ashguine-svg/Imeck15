Imeck15 Leitfaden für Klick-Einstellungen pro Bild

Dieser Leitfaden erklärt, wie Klick-Aktionen konfiguriert werden, wenn ein registriertes Bild gefunden wird. Er ist hauptsächlich für die Automatisierung von Vorgängen in Fensteranwendungen gedacht.

1. So registrieren Sie Bilder zum Klicken

Registrieren Sie zuerst das Bild, das als Klick-Ziel dienen soll. Die Verwendung der Funktion "Bild aufnehmen" ist die grundlegende Methode.
Tipp: Es wird empfohlen, kleine Teile wie Schaltflächen oder Symbole als Ziel zu erfassen.

  • Grund 1: Schnellere Verarbeitung
    Die Suche nach einem kleinen Bild ist weniger anspruchsvoll für den PC als die Suche auf dem gesamten Bildschirm.
  • Grund 2: Genaues Klicken
    Selbst wenn mehrere identische Schaltflächen auf dem Bildschirm existieren, ermöglicht das Erfassen nur eines bestimmten Teils ein präzises Klicken auf das beabsichtigte Ziel.

2. So steuern Sie die Klick-Reihenfolge (Intervall-Einstellung)

Das "Intervall" ist die minimale Wartezeit (in Sekunden) nach dem Klicken eines Bildes, bevor dasselbe Bild erneut geklickt werden kann.

[Wichtig] Klick-Prioritätsmechanismus
Wenn während der Überwachung mehrere klickbare Bilder auf dem Bildschirm gefunden werden, priorisiert Imeck15 das Klicken auf das Bild mit der kürzesten "Intervall"-Einstellung und klickt nicht auf andere Bilder. Verwenden Sie diesen Mechanismus, um die Klick-Sequenz zu steuern.

3. Technik zum Klicken auf mehrere Stellen auf einem Bildschirm

Durch Anwendung des "Intervall-Einstellungs"-Mechanismus können Sie festlegen, dass nacheinander auf mehrere Stellen auf einem einzigen Bildschirm geklickt wird.
Voraussetzung: Das Bild oder der Text an der geklickten Stelle muss sich nach dem Klicken ändern (verschwinden, ausgegraut werden usw.).

Beispiel-Einrichtungsschritte:

  1. Registrieren Sie die Teile A, B und C, die Sie auf dem Bildschirm anklicken möchten, als separate Bilder.
  2. Stellen Sie kürzere Intervallzeiten für Bilder ein, die Sie früher anklicken möchten. (z. B. A: 1.5 Sek., B: 2.0 Sek., C: 2.5 Sek.)
  3. Starten Sie die Überwachung. Zuerst wird Teil A mit dem kürzesten Intervall angeklickt.
  4. Nach dem Klicken wird Teil B zum nächsten Klick-Ziel im folgenden Überwachungszyklus, falls Teil A vom Bildschirm verschwindet.
  5. Ebenso wird Teil C zuletzt angeklickt.

4. So passen Sie die Erkennungsgenauigkeit und den Bereich an

  • Erkennungsschwelle:
    Diese Einstellung bestimmt, wie ähnlich Bilder sein müssen, um als "Übereinstimmung" zu gelten. Ein Wert um 0.8 ist normalerweise ausreichend, aber das Einstellen auf 0.9 oder höher kann effektiv sein, um geringfügige Unterschiede wie Textvariationen streng zu unterscheiden.
  • Suchbereich (ROI-Einstellung):
    Das Aktivieren von "ROI aktiviert" beschränkt das Suchziel auf einen 200x200 Pixel großen Bereich, der um die Klick-Koordinate zentriert ist. Dies beschleunigt die Verarbeitung erheblich und reduziert die Last auf den PC stark. (Der Variable ROI-Modus ermöglicht das freie Einstellen des Bereichs).
  • ROI-Bereichsspezifikation:
    Anstelle eines 200x200 Bereichs, der um die Klick-Position zentriert ist, ermöglicht diese Funktion das Festlegen eines spezifischen Bereichs, auf den man sich konzentrieren möchte, und das Bestimmen einer separaten Klick-Position.

5. "Debounce"-Einstellung für spezielle Situationen

"Debounce" wird für komplexe Vorgänge wie "Bild A mit einem kurzen Intervall klicken, dann ein anderes Bild B klicken und nach einer kurzen Pause Bild A erneut klicken, um zum nächsten Bildschirm zu gelangen" verwendet.

Mechanismus: Das Festlegen einer Debounce-Zeit verlängert die Wartezeit für den zweiten Klick auf "Intervall + Debounce-Zeit", wenn dasselbe Bild nacheinander als Ziel ausgewählt wird. Dies schafft eine Gelegenheit, dass andere Bilder zuerst angeklickt werden.
