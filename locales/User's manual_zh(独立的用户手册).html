1. 简介

Imeck15 是什么？

Imeck15 是一款能够识别 PC 屏幕上显示的特定图像，并自动点击预设位置的工具。主要用于自动化 PC 游戏和应用程序中的例行重复性任务。

主要功能

  • 图像识别：检测已注册图像（模板）何时出现在屏幕上。
  • 自动点击：在检测到图像时自动点击指定的坐标或区域。
  • 灵活设置：允许为每个图像详细配置，如点击间隔、识别精度和点击位置。
  • 性能调整：配备多种选项，可根据操作环境调整处理速度和 PC 负载。
  • 文件夹功能：在文件夹中管理图像，实现基于组的操作控制。
  • 拖放操作：直观地重新排列图像列表中的项目或将其移动到文件夹中。

2. 基本用法

以下是开始使用 Imeck15 的四个基本步骤。

步骤 1：注册图像

注册将作为自动点击触发器的图像。

  1. 按下主屏幕上的 [捕获图像] 按钮。
  2. 当出现“识别区域设置”对话框时，选择 [设置矩形]。
  3. 屏幕将被半透明覆盖层覆盖。拖动鼠标圈出要点击的目标（如按钮或图标）。尽量使选择范围尽可能小，同时仍能捕捉到独特的特征。
  4. 当出现文件名输入对话框时，给它一个描述性的名称并保存。
  5. 您刚刚注册的图像将被添加到左侧的“已注册图像”列表中。

步骤 2：设置识别区域

指定 Imeck15 搜索图像的区域。限制此区域可显著降低 PC 负载。

  1. 按下主屏幕顶部的 [设置识别区域] 按钮。
  2. [设置矩形]：如果您只想在整个屏幕的特定区域内搜索，请选择此项。
  3. [设置窗口]：如果您只想在特定的应用程序窗口内搜索，请选择此项。选择窗口时，它会根据其标准尺寸自动计算缩放因子，从而提高图像识别精度（详见下文）。

步骤 3：设置点击位置

配置在找到已注册图像时点击的位置。

  1. 从左侧列表中选择所需的图像。
  2. 图像将显示在右侧的“图像预览”选项卡中。
  3. 在底部的“单个图像设置”区域中，勾选 [点点击] 或 [范围点击] 复选框。
  4. 当鼠标悬停在预览图像上时，光标将变为十字准线。
    • 点点击：点击您想要定位的确切位置。它将用红点标记。
    • 范围点击：拖动鼠标定义一个矩形区域。它将显示为一个蓝色框。点击将发生在此区域内的某个位置（也可以设置为随机）。

步骤 4：开始和停止监控

设置完成后，开始自动点击。

  • 开始：按下主屏幕左上角的 [开始监控] 按钮。状态将变为“监控中...”
  • 停止：按下 [停止监控] 按钮。
快捷键（鼠标）：
  • 右键双击：停止监控
  • 右键三击：开始监控

3. 主屏幕功能

主控制区域

窗口顶部的操作区域，始终可见。

  • [开始监控]/[停止监控]：切换自动点击的开关。
  • [性能]：打开“性能监视器”，您可以在其中查看 CPU 使用率、FPS 等。
  • [设置识别区域]：设置搜索图像的区域。
  • [最小化 UI 模式]：切换到一个只有基本控件的小窗口。
  • [图像文件夹]：在资源管理器中打开存储已注册图像的文件夹。
  • 状态显示：显示当前状态（待机中、监控中...）。

已注册图像区域（左侧）

用于管理已注册图像和文件夹的区域。

  • 图像列表：
    • 已注册的图像和文件夹以树形格式显示。
    • 您可以拖放项目以更改其顺序或将其移动到文件夹中。
    • 您可以通过按住 Shift 或 Ctrl 键单击来选择多个项目。
  • [▲ 上移] / [▼ 下移]：将所选项目在列表中向上或向下移动。
  • [添加图像]：将现有的图像文件（png、jpg 等）添加到列表中。
  • [捕获图像]：捕获屏幕的新部分进行注册。
  • [删除所选]：删除当前选定的项目。
  • [创建文件夹]：创建一个文件夹来组织图像。
  • [移入/移出文件夹]：将图像移入或移出文件夹。
  • 右键单击文件夹：右键单击文件夹会打开一个菜单来设置其行为模式。

设置与预览区域（右侧）

通过切换选项卡访问各种信息和设置。

  • 图像预览：允许预览所选图像并设置点击位置。
  • 识别区域：显示当前设置的识别区域的预览。
  • 日志：显示操作日志。
  • 自动缩放：根据窗口大小配置图像缩放设置。
  • 应用设置：配置 Imeck15 的整体行为。
  • 使用方法：显示关于单个图像设置的详细指南。
  • 单个图像设置（底部）：配置在左侧列表中选择的图像的单独设置。

4. 详细功能与技巧

4.1. 设置识别区域的好处

使用 [设置窗口] 定义识别区域除了减少 PC 负载外，还提供了两大好处。

1. 基于窗口缩放的自动尺寸调整

即使目标窗口的大小发生变化，此功能也能通过自动计算模板图像的比例来保持识别精度。

工作原理：
  • 存储基础尺寸：第一次指定特定窗口时，系统会提示您将其当前尺寸保存为“基础尺寸（1.0倍比例）”。
  • 自动计算比例：随后，当您指定同一窗口时，它会将基础尺寸与当前尺寸进行比较，并自动计算缩放因子（例如 1.2 倍）。
  • 高精度匹配：将此计算出的比例应用于模板匹配，确保即使窗口大小发生变化也能准确识别。

2. 点击前自动激活窗口

在执行点击之前，此功能会检查目标窗口是否在前台（活动状态），如果不是，则自动激活它。

工作原理：
  • 活动状态检查：在点击之前立即验证目标窗口是否为活动的前台窗口。
  • 强制激活：如果窗口隐藏在另一个窗口后面或已最小化，它会在执行点击之前恢复窗口并将其置于最前面。
  • 防止误击：此功能可防止意外点击其他窗口，大大提高了自动化过程的可靠性。

4.2. 单个图像设置项

为列表中选择的图像配置单独的行为。

  • 识别精度（阈值）：设置图像必须有多相似才能被视为“匹配”。0.8 是标准值；增加该值会使检查更严格（1.0 是完美匹配）。
  • 启用 ROI（限制搜索区域）：开启后，仅搜索一个小的、已定义的区域，从而加快处理速度。
    • 固定：自动将以点击坐标为中心的 200x200 像素区域设置为 ROI。
    • 可变：允许您通过在预览屏幕上拖动来自由设置搜索区域 (ROI)。按下 [设置 ROI 区域] 按钮可在预览上启用区域绘制。
  • 备用点击：作为最后手段。如果在指定持续时间（秒）内未找到其他正常图像，则将单击此图像。
  • 间隔（秒）：单击后，再次单击*同一图像*之前的最短等待时间。此值决定了点击优先级（值越小优先级越高）。
  • 防抖（秒）：如果连续定位同一图像进行点击，则将第二次点击的等待时间延长至“间隔 + 防抖时间”。这为点击其他图像创造了机会。
  • 点击类型：在 [点点击]、[范围点击] 和 [范围内随机] 之间选择。

4.3. 如何控制点击顺序

如果屏幕上同时找到多个图像，Imeck15 会优先点击具有最小“间隔”值的图像。您可以使用此机制来控制点击优先级顺序。

示例：

  • 将图像 A 的间隔设置为 1.5
  • 将图像 B 的间隔设置为 2.0

→ 即使 A 和 B 同时出现在屏幕上，图像 A（间隔较短）也总是会被先点击。

高级技巧：
在一个屏幕上有多个按钮，这些按钮在被点击后会改变状态（消失、变灰等），您可以通过按照希望点击的顺序设置间隔（例如 1.5、2.0、2.5...）来自动化一系列操作。

4.4. 利用文件夹功能

将图像整理到文件夹中，可以按组控制它们的行为。右键单击文件夹以打开其设置窗口。

  • 正常：文件夹中的图像按正常方式搜索。
  • 停止搜索（排除）：文件夹中的所有图像都将从搜索中排除。用于临时禁用一组图像。
  • 定时器优先：经过指定时间（分钟）后，模式切换为*仅*搜索此文件夹中的图像。用于对仅应按特定间隔运行的一系列操作进行分组。

图像识别优先模式

这是一个强大的功能，用于自动化响应特定情况（例如弹出窗口）的一系列操作。

工作原理：

  1. 触发：如果在屏幕上找到属于此文件夹的至少一个图像，则启动优先模式。
  2. 行为：在优先模式下，仅搜索并点击此文件夹中的图像。
  3. 停用条件：当满足以下任一条件时，优先模式会自动停用并返回正常监控：
    • 文件夹中的所有图像都至少被点击过一次（完成）。
    • 在指定的超时持续时间（秒）内未找到来自该文件夹的图像。
用法示例：
考虑一个游戏中出现的“库存已满”窗口。
  1. 捕获所有相关图像：“库存已满”标题、“确定”按钮、“取消”按钮等。
  2. 将这些图像放入一个名为“处理库存已满”的新文件夹中，并将其模式设置为“图像识别优先”。
  3. 调整文件夹内图像的间隔，使“确定”按钮最后被点击。
通过这种设置，“处理库存已满”文件夹通常会被忽略。但是，一旦库存窗口出现，Imeck15 就会检测到它，自动执行必要的操作（如点击“确定”），完成后返回正常监控。

4.5. 全局应用程序设置

在“应用设置”选项卡中调整 Imeck15 的整体行为。

  • 灰度匹配：以黑白方式比较图像。速度更快，但无法区分颜色。
  • 使用 DXCam：一种更快的屏幕捕获方法。对捕获游戏屏幕有效，但可能不适用于所有环境。
  • 跳帧：较高的值会降低屏幕检查频率，从而减少 PC 负载，但也会减慢响应时间。
  • 使用 OpenCL（GPU 加速）：如果存在兼容的 GPU，可以加快图像处理速度。
  • 屏幕稳定性检查：检测屏幕渲染或效果，并在屏幕稳定之前暂停点击。调整阈值以更改检查的灵敏度。
  • 轻量模式：通过降低屏幕捕获分辨率来大幅减少 PC 负载。您可以从预设（标准、性能、极限）中进行选择。

5. 性能优化

如果您觉得 Imeck15 运行缓慢或导致您的 PC 卡顿，请检查以下设置。

基本优化步骤

按效果顺序列出。

  1. 限制识别区域：这是最有效的方法。使用窗口或矩形设置将搜索区域缩小到最低限度。
  2. 启用轻量模式：在“应用设置”选项卡中启用此功能。它可以显著改善 PC 负载。
  3. 启用单个图像 ROI：这使得处理速度非常快，因为应用程序仅在点击目标周围的小区域内搜索。
  4. 增加跳帧值：这是与响应速度的权衡，但会减少负载。
  5. 使用 DXCam：如果您的环境支持，这可以提供快速、低 CPU 负载的捕获。

关于节能模式

“应用设置”选项卡中的[节能模式]是一项进一步降低 PC 负载的功能。

工作原理：
  • 激活：在最后一次点击后 5 秒内未找到可识别图像时自动激活。
  • 行为：激活后，屏幕检查频率降至大约每秒一次，显著降低 CPU 使用率。
  • 停用：如果在节能模式期间识别到图像并执行了点击，则会自动停用并返回正常的监控频率。
此功能在自动化等待时间较长的任务时特别有用。默认情况下启用，但如果您始终需要最快的响应速度，可以将其关闭。

6. 各种窗口

性能监视器

通过主屏幕上的 [性能] 按钮打开。它允许您实时查看 CPU/内存使用率、FPS（每秒帧检查次数）、总点击次数和其他信息。您还可以使用顶部的按钮开始/停止监控或设置识别区域。可以使用快捷键：右键双击停止监控，右键三击开始监控。

最小化 UI 模式

通过 [最小化 UI 模式] 按钮切换。这将隐藏主窗口并显示一个带有基本控件（如开始/停止监控）的小窗口。

7. 常见问题解答 (FAQ)

问：图像无法正确找到。

答：请检查以下几点：

  • “识别区域”是否设置正确？
  • “单个图像设置”中的“识别精度”是否设置得太高？尝试稍微降低它。
  • 您的游戏或应用程序的窗口大小是否已更改？检查“自动缩放”选项卡中的设置或尝试重新捕获图像。

问：应用程序很卡，我的电脑运行缓慢。

答：请参阅上面的“5. 性能优化”部分并检查您的设置。

问：它在屏幕转换过程中点击。

答：检查“应用设置”选项卡中的“屏幕稳定性检查”是否已启用。如果是，请尝试稍微降低“阈值”值，使检查更严格。
