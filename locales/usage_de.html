Imeck15 Anleitung zur Klickeinstellung pro Bild

Diese Anleitung erklärt, wie Klick-Aktionen konfiguriert werden, wenn ein registriertes Bild gefunden wird. Sie ist primär für die Automatisierung von Vorgängen in Fenster-Anwendungen gedacht.

1. Wie man Bilder für den Klick registriert

Registrieren Sie zuerst das Bild, das als Klick-Ziel dienen soll. Die Verwendung der Funktion „Bild aufnehmen“ ist die grundlegende Methode.
Tipp: Es wird empfohlen, kleine Teile wie Schaltflächen oder Symbole als Ziel zu erfassen.

  • Grund 1: Schnellere Verarbeitung
    Die Suche nach einem kleinen Bild ist weniger anspruchsvoll für den PC als die Suche auf dem gesamten Bildschirm.
  • Grund 2: Präzises Klicken
    Selbst wenn mehrere identische Schaltflächen auf dem Bildschirm existieren, erlaubt das Erfassen nur eines bestimmten Teils das präzise Klicken auf das beabsichtigte Ziel.

2. Wie man die Klick-Reihenfolge steuert (Intervall-Einstellung)

Das „Intervall“ ist die minimale Wartezeit (in Sekunden), nachdem auf ein Bild geklickt wurde, bevor dasselbe Bild erneut angeklickt werden kann.

[Wichtig] Klick-Prioritäts-Mechanismus
Wenn während der Überwachung mehrere klickbare Bilder auf dem Bildschirm gefunden werden, priorisiert Imeck15 das Klicken auf das Bild mit der kürzesten „Intervall“-Einstellung und klickt nicht auf andere Bilder. Verwenden Sie diesen Mechanismus, um die Klick-Sequenz zu steuern.

3. Technik für das Klicken an mehreren Stellen auf einem Bildschirm

Durch Anwendung des „Intervall-Einstellungs“-Mechanismus können Sie festlegen, dass nacheinander auf mehrere Stellen auf einem einzigen Bildschirm geklickt wird.
Voraussetzung: Das Bild oder der Text an der angeklickten Stelle muss sich nach dem Klicken ändern (verschwinden, ausgegraut werden usw.).

Beispiel-Einrichtungsschritte:

  1. Registrieren Sie die Teile A, B und C, die Sie auf dem Bildschirm anklicken möchten, als separate Bilder.
  2. Legen Sie kürzere Intervallzeiten für Bilder fest, die Sie früher anklicken möchten. (z. B. A: 1.5 Sek., B: 2.0 Sek., C: 2.5 Sek.)
  3. Starten Sie die Überwachung. Zuerst wird Teil A mit dem kürzesten Intervall angeklickt.
  4. Nach dem Klicken, wenn Teil A vom Bildschirm verschwindet, wird Teil B im nachfolgenden Überwachungszyklus zum nächsten Klick-Ziel.
  5. Ebenso wird Teil C zuletzt angeklickt.

4. Wie man die Erkennungsgenauigkeit und den Bereich anpasst

  • Erkennungsschwellenwert:
    Diese Einstellung bestimmt, wie ähnlich Bilder sein müssen, um als „Übereinstimmung“ betrachtet zu werden. Ein Wert um 0.8 ist normalerweise ausreichend, aber das Setzen auf 0.9 oder höher kann effektiv sein, um geringfügige Unterschiede wie Textvariationen streng zu unterscheiden.
  • Suchbereich (ROI-Einstellung):
    Das Aktivieren von „ROI aktivieren“ beschränkt das Suchziel auf einen 200x200 Pixel großen Bereich, zentriert auf der Klick-Koordinate. Dies beschleunigt die Verarbeitung erheblich und reduziert die Last auf dem PC stark. (Der variable ROI-Modus erlaubt das freie Festlegen des Bereichs).
  • ROI-Bereichs-Spezifikation:
    Anstelle eines 200x200-Bereichs, zentriert auf der Klick-Position, erlaubt diese Funktion das Spezifizieren eines bestimmten Bereichs, auf den fokussiert werden soll, und das Festlegen einer separaten Klick-Position.

5. „Entprellung“ (Debounce) Einstellung für spezielle Situationen

„Entprellung“ wird für komplexe Vorgänge verwendet, wie z. B. „Klick auf Bild A mit kurzem Intervall, dann Klick auf ein anderes Bild B und nach einer kurzen Pause Klick auf Bild A erneut, um zum nächsten Bildschirm fortzufahren“.

Mechanismus: Das Festlegen einer Entprellzeit verlängert die Wartezeit für den zweiten Klick auf „Intervall + Entprellzeit“, wenn dasselbe Bild nacheinander als Ziel festgelegt wird. Dies schafft eine Gelegenheit, dass andere Bilder zuerst geklickt werden.
