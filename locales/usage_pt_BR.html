Guia de Configurações de Clique por Imagem do Imeck15

Este guia explica como configurar as ações de clique quando uma imagem registrada é encontrada. É destinado principalmente para automatizar operações em aplicativos de janela.

1. Como Registrar Imagens para Clicar

Primeiro, registre a imagem que servirá como alvo do clique. Usar a função "Capturar Imagem" é o método básico.
Dica: Recomenda-se capturar partes pequenas como botões ou ícones como alvo.

  • Razão 1: Processamento Mais Rápido
    Buscar uma imagem pequena exige menos do PC do que buscar a tela inteira.
  • Razão 2: Clique Preciso
    Mesmo que existam vários botões idênticos na tela, capturar apenas uma parte específica permite um clique preciso no alvo pretendido.

2. Como Controlar a Ordem dos Cliques (Configuração de Intervalo)

O "Intervalo" é o tempo mínimo de espera (em segundos) após clicar em uma imagem, antes que a mesma imagem possa ser clicada novamente.

[Importante] Mecanismo de Prioridade de Clique
Se várias imagens clicáveis forem encontradas na tela durante o monitoramento, o Imeck15 prioriza clicar naquela com a configuração de "Intervalo" mais curta e não clica em outras imagens. Use este mecanismo para controlar a sequência de cliques.

3. Técnica para Clicar em Múltiplos Locais em uma Única Tela

Ao aplicar o mecanismo de "Configuração de Intervalo", você pode fazer com que ele clique em múltiplos locais sequencialmente em uma única tela.
Pré-requisito: A imagem ou texto no local clicado deve mudar (desaparecer, ficar cinza, etc.) após ser clicado.

Exemplo de Passos de Configuração:

  1. Registre as partes A, B e C que você deseja clicar na tela como imagens separadas.
  2. Defina tempos de intervalo mais curtos para as imagens que você deseja clicar mais cedo. (ex: A: 1.5 seg, B: 2.0 seg, C: 2.5 seg)
  3. Inicie o monitoramento. Primeiro, a parte A com o intervalo mais curto será clicada.
  4. Após o clique, se a parte A desaparecer da tela, a parte B se tornará o próximo alvo de clique no ciclo de monitoramento subsequente.
  5. Da mesma forma, a parte C será clicada por último.

4. Como Ajustar a Precisão e o Intervalo de Reconhecimento

  • Limite de Reconhecimento:
    Esta configuração determina o quão semelhantes as imagens precisam ser para serem consideradas uma "correspondência". Um valor em torno de 0.8 geralmente é suficiente, mas configurar para 0.9 ou superior pode ser eficaz para distinguir estritamente pequenas diferenças, como variações de texto.
  • Intervalo de Busca (Configuração ROI):
    Habilitar "ROI Ativado" limita o alvo de busca a uma área de 200x200 pixels centrada na coordenada do clique. Isso acelera significativamente o processamento e reduz muito a carga da PC. (O modo ROI Variável permite definir o intervalo livremente).
  • Especificação do Intervalo ROI:
    Em vez de um intervalo de 200x200 centrado na posição de clique, este recurso permite que você especifique uma área específica para focar e designar uma posição de clique separada.

5. Configuração de "Debounce" para Situações Especiais

"Debounce" é usado para operações complexas como "clicar na imagem A com um intervalo curto, depois clicar em outra imagem B e, após uma breve pausa, clicar na imagem A novamente para prosseguir para a próxima tela".

Mecanismo: Configurar um tempo de debounce estende o período de espera para o segundo clique para "Intervalo + Tempo de Debounce" quando a mesma imagem é alvo consecutivamente. Isso cria uma oportunidade para que outras imagens sejam clicadas primeiro.
